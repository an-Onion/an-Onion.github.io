Alert Pattern 示例

使用 daisyUI 组件库构建无障碍通知组件

基本 Alert 通知

使用 role="alert" 实现无障碍通知

ℹ️ 系统将在今晚 10 点进行维护，届时服务将暂停 2 小时。 

<div role="alert" class="alert">
  <span>ℹ️</span>
  <span>系统将在今晚 10 点进行维护...</span>
</div>

信息 Alert

使用 alert-info 类表示信息性通知

ℹ️ 有 12 条未读消息，点击查看。 

<div role="alert" class="alert alert-info">
  <span>ℹ️</span>
  <span>有 12 条未读消息...</span>
</div>

成功 Alert

使用 alert-success 类表示操作成功

✅ 您的订单已确认，订单号为 #12345。 

<div role="alert" class="alert alert-success">
  <span>✅</span>
  <span>您的订单已确认...</span>
</div>

警告 Alert

使用 alert-warning 类表示需要用户注意

⚠️

库存不足

您选择的商品仅剩 3 件，建议您尽快下单。


<div role="alert" class="alert alert-warning">
  <span>⚠️</span>
  <div>
    <h3>库存不足</h3>
    <p>您选择的商品仅剩 3 件...</p>
  </div>
</div>

错误 Alert

使用 alert-error 类表示错误或失败

❌ 保存失败：无法连接到服务器，请检查您的网络连接。 

<div role="alert" class="alert alert-error">
  <span>❌</span>
  <span>保存失败：无法连接到服务器...</span>
</div>

Soft 样式 Alert

使用 alert-soft 类实现柔和样式

ℹ️ 有 12 条未读消息，点击查看。
✅ 您的购买已确认！
⚠️ 警告：无效的电子邮件地址！
❌ 错误！任务执行失败。 

<div role="alert" class="alert alert-info alert-soft">
  <span>ℹ️</span>
  <span>有 12 条未读消息...</span>
</div>

动态 Alert 示例

表单提交时动态显示 Alert 


<template id="alert-success-template">
  <div role="alert" class="alert alert-success">
    <span>✅</span>
    <span>表单提交成功！我们将在 24 小时内回复您。</span>
  </div>
</template>

<form id="contact-form">
  <input type="email" id="email" required>
  <button type="submit">提交</button>
</form>

<div id="form-feedback"></div>

<script>
form.addEventListener('submit', function(e) {
  e.preventDefault();
  const email = document.getElementById('email').value;
  
  feedback.innerHTML = '';
  
  if (!email.includes('@')) {
    const template = document.getElementById('alert-error-template');
    feedback.appendChild(template.content.cloneNode(true));
  } else {
    const template = document.getElementById('alert-success-template');
    feedback.appendChild(template.content.cloneNode(true));
  }
});
</script>
