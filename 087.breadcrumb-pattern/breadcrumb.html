Breadcrumb Pattern 示例

使用 daisyUI 组件库构建无障碍面包屑导航

基础面包屑导航

使用 nav 元素包裹，aria-label 标记导航区域

  • 首页
  • 产品
  • 笔记本电脑
<nav aria-label="面包屑导航">
  <ul>
    <li><a href="/">首页</a></li>
    <li><a href="/products/">产品</a></li>
    <li aria-current="page">笔记本电脑</li>
  </ul>
</nav>

带分隔符的面包屑导航

daisyUI 自动添加分隔符，使用 aria-hidden 隐藏辅助技术

  • Home
  • Documents
  • Add Document
<nav aria-label="面包屑导航">
  <ul>
    <li><a href="/">首页</a></li>
    <li aria-hidden="true">/</li>
    <li><a href="/docs/">文档</a></li>
    <li aria-hidden="true">/</li>
    <li aria-current="page">快速入门</li>
  </ul>
</nav>

带图标的面包屑导航

为每个导航项添加 SVG 图标增强可识别性

  • 首页
  • 商店
  • 商品详情 
<nav aria-label="面包屑导航">
  <ul>
    <li>
      <a href="/">
        <svg class="h-4 w-4 stroke-current">...</svg>
        首页
      </a>
    </li>
    <li>
      <a href="/shop/">
        <svg class="h-4 w-4 stroke-current">...</svg>
        商店
      </a>
    </li>
    <li aria-current="page">
      <span class="inline-flex items-center gap-2">
        <svg class="h-4 w-4 stroke-current">...</svg>
        商品详情
      </span>
    </li>
  </ul>
</nav>

电商网站产品页面包屑导航

典型的电商场景，展示从首页到产品详情的完整路径

  • 首页
  • 家用电器
  • 厨房电器
  • 咖啡机
  • 全自动咖啡机 X2000
<nav aria-label="商品位置" class="breadcrumbs">
  <ul>
    <li><a href="https://www.example.com/">首页</a></li>
    <li><a href="https://www.example.com/electronics/">家用电器</a></li>
    <li><a href="https://www.example.com/electronics/kitchen/">厨房电器</a></li>
    <li><a href="https://www.example.com/electronics/kitchen/coffee/">咖啡机</a></li>
    <li aria-current="page">全自动咖啡机 X2000</li>
  </ul>
</nav>
